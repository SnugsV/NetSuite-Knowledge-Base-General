# Manufacturing KPIs

## Quick Summary

Manufacturing KPIs measure production efficiency, cost control, quality, and schedule adherence. Regular monitoring identifies bottlenecks and improvement opportunities.

## Key Performance Indicators

### Production Efficiency

| KPI | Calculation | Target |
|---|---|---|
| **Schedule Adherence** | % of work orders completed on time | > 95% |
| **Cycle Time** | Average time from work order release to completion | Decreasing |
| **Throughput** | Units produced per period | Increasing |
| **Overall Equipment Effectiveness (OEE)** | Availability × Performance × Quality | > 85% |

### Cost Metrics

| KPI | Calculation | Target |
|---|---|---|
| **Unit Cost Variance** | (Actual - Standard) / Standard × 100 | ± 5% |
| **Material Variances** | Sum of all material price/usage variances | Tracking |
| **Labor Efficiency** | Standard hours / actual hours | > 90% |
| **Overhead Absorption** | Applied overhead / actual overhead | 100% ± 2% |

### Quality Metrics

| KPI | Calculation | Target |
|---|---|---|
| **First Pass Yield** | Units passing inspection on first attempt / total units | > 95% |
| **Scrap Rate** | Scrapped units / total units produced | < 2% |
| **Rework Rate** | Units requiring rework / total units | < 3% |

### Inventory Metrics

| KPI | Calculation | Target |
|---|---|---|
| **Raw Material Turns** | Cost of materials used / average raw material value | Industry dependent |
| **WIP Turns** | Cost of goods manufactured / average WIP value | Industry dependent |
| **WIP Aging** | Age of oldest open work orders | < 30 days |
