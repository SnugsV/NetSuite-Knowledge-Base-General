# Administration

This module covers NetSuite administration — the configuration, security, and maintenance tasks that keep a NetSuite account running effectively.

## Module Overview

Administration is the most consequential role in NetSuite. Configuration decisions made during setup affect every user, every transaction, and every report. Mistakes — misconfigured permissions, incorrectly enabled features, overlooked security settings — can take months to remediate.

This module teaches administration from first principles. It assumes you have completed the Getting Started path or have equivalent foundational knowledge.
