# Period Close

## Quick Summary

The period close is the process of finalizing all transactions for a defined accounting period, making adjusting entries, reconciling accounts, and locking the period so no further changes can be made.

## Difficulty

Advanced

## Estimated Time

15 minutes

## Business Question

"How do I ensure that every transaction for this month is correctly recorded before I finalize the books?"

## Overview

The period close is the most critical recurring process in accounting. Every transaction from every operational module must be captured, reconciled, and verified before financial statements can be produced.

## The Close Process

```
Step 1: Confirm all transactions are posted
Step 2: Reconcile subledgers (AP, AR, Inventory)
Step 3: Record adjusting entries (accruals, depreciation, prepaids)
Step 4: Reconcile bank accounts
Step 5: Review trial balance for anomalies
Step 6: Run financial statements
Step 7: Management review
Step 8: Lock the period
```

## Common Issues

| Issue | Resolution |
|---|---|
| **Unposted transactions** | Find and post before closing |
| **Trial balance out of balance** | Review journal entries, system postings |
| **Subledger doesn't match GL** | Run subledger aging, find discrepancies |
| **Unreconciled bank accounts** | Complete bank reconciliation |
| **Inventory adjustments not recorded** | Post all adjustments before close |
