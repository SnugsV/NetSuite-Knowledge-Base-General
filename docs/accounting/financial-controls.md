# Financial Controls

## Quick Summary

Financial controls are policies, procedures, and system configurations that protect assets, ensure accurate reporting, and prevent fraud. Effective controls are essential for audit readiness and regulatory compliance.

## Difficulty

Intermediate

## Estimated Time

10 minutes

## Overview

Financial controls exist because errors and fraud happen. A simple data entry mistake can misstate financial results. Intentional fraud can destroy a company. Controls reduce risk by preventing, detecting, and correcting errors.

## Key Controls

| Control Area | Example Controls |
|---|---|
| **Segregation of duties** | PO creation ≠ PO approval ≠ check signing |
| **Transaction approvals** | Journal entries require manager approval |
| **Access controls** | Users have only needed permissions |
| **Reconciliation** | Bank accounts reconciled monthly |
| **Period close checklist** | Required steps before closing period |
| **Audit trail review** | Monitor unusual transactions |

## Red Flags

| Red Flag | Potential Issue |
|---|---|
| Transactions posted by same person who approved | Segregation of duties violation |
| Journal entries posted after period close | Period close control failure |
| AP checks created without PO | Procure-to-pay control bypass |
| Manual journal entries increasing each month | System configuration gap |
| High volume of credit memos | Revenue reporting risk |
