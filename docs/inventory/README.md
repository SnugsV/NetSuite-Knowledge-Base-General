# Inventory

This module covers Inventory Management in NetSuite — the systems, processes, and configurations for tracking, valuing, and controlling stock.

## About This Module

Inventory is the operational center of NetSuite. Every other module touches it: **Purchasing** creates inventory, **Sales** consumes it, **Manufacturing** transforms it, **Accounting** values it, and **Reporting** monitors it. Understanding Inventory means understanding how NetSuite works as a connected ERP system.

## Release 0.5A — Inventory Fundamentals

This release covers the foundational concepts. It is designed for readers who have completed the Getting Started path and want to understand what inventory is, how NetSuite models it, and how it connects to the broader ERP ecosystem.

### Learning Objectives

After completing this module, you will be able to:

- Explain what inventory management is and why companies do it
- Read and understand the quantity fields in NetSuite (On Hand, Available, Committed, On Order)
- Describe how inventory valuation affects financial statements
- Create item records and choose the correct item type
- Set up locations for single and multi-location inventory
- Configure units of measure and understand conversions
- Explain how inventory connects to Purchasing, Sales, Manufacturing, and Accounting

### Skills Gained

- Item record configuration
- Understanding inventory data model
- Basic inventory transactions terminology
- ERP integration awareness

### Business Scenarios Covered

- Retail store inventory tracking
- Distribution warehouse operations
- Manufacturing component management
- Multi-location stock management
- Service company stocked items

### What You Will Not Learn Here

This release covers fundamentals only. Operational workflows — transfers, adjustments, receipts, fulfillments, replenishment — are covered in **Release 0.5B — Inventory Operations** (planned).
