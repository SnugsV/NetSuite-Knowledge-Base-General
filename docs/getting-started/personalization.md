# Personalization

## Quick Summary

NetSuite allows users to personalize their experience without administrator involvement. You can customize your dashboard, center layout, saved searches, preferences, and home page portlets to match how you work.

## Difficulty

Beginner

## Estimated Time

10 minutes

## Overview

Personalization in NetSuite is the ability to tailor the interface and functionality to your preferences without needing system-level permissions. Personalization typically affects only your own view and does not change what other users see.

There are several levels of personalization:

1. **Dashboard customization** — Add, remove, and arrange portlets
2. **Center personalization** — Choose which center is your default landing page
3. **Saved search personalization** — Create personal saved searches that only you can see
4. **User preferences** — Set defaults for language, time zone, date format, and communication preferences
5. **Home page personalization** — Configure the Home Center dashboard layout

## Dashboard Personalization

You can customize your dashboard in any center that you have access to. Common customizations include:

- **Adding portlets**: Add KPIs, saved searches, reminders, calendars, and charts
- **Removing portlets**: Remove portlets you do not use
- **Rearranging portlets**: Drag portlets to change their position and size
- **Configuring portlet settings**: Set filters, layout options, and refresh intervals for each portlet

To personalize your dashboard:

1. Go to the dashboard you want to customize
2. Click **Personalize Dashboard** (or similar link, depending on theme)
3. Use the dashboard customization page to add, remove, or arrange portlets
4. Save your changes

## Saved Search Personalization

When you create a saved search, you can control who can see it:

- **Personal**: Only you can see and run the search
- **Shared**: Users with the appropriate role can see it
- **Public**: All users with permissions can see it

Creating personal saved searches is a common way to build your own monitoring tools. For example, you might create a personal saved search for "Orders I entered today" or "Items I need to reorder."

See the [Saved Searches module](../saved-searches/overview.md) for more detail.

## User Preferences

User preferences control how NetSuite behaves for your account. To access preferences:

```text
Home > Set Preferences
```

Common preferences you can set:

| Preference | Options |
|---|---|
| Language | Choose your language |
| Time Zone | Set your local time zone |
| Date Format | Month/Day/Year or Day/Month/Year |
| Number Format | Decimal separator, grouping |
| Email | Default email address for notifications |
| Appearance | Theme, color scheme, font size |
| Home Page | Default center on login |

## Form Personalization

While customizing forms (adding/removing fields, rearranging tabs) typically requires administrator-level permissions, some form personalization options may be available depending on your role and account configuration.

If you want a form customized for your role, speak with your NetSuite administrator.

## Center Personalization

Some centers allow you to choose which sub-tabs or views appear first. For example, you may be able to set the default view for a dashboard or choose which portlet appears at the top.

## Common Mistakes

- **Not personalizing at all**: Many users leave the default dashboard unchanged, missing the opportunity to build a daily workflow that saves time.
- **Over-personalizing**: Adding too many portlets or saved searches creates noise. Focus on what you actually check daily.
- **Assuming personalization changes affect other users**: Dashboard changes are typically personal. You are not altering the experience for your team.
- **Not knowing the difference between personal and shared saved searches**: A personal saved search is not visible to your team unless you intentionally share it.

## Best Practices

- Start your first day by personalizing your Home dashboard with the portlets you will use most often.
- Set your user preferences for time zone, date format, and language on day one.
- Create personal saved searches for the reports you run more than once per week.
- Review your personalization quarterly as your role evolves.
- Learn the available portlet types — you may discover dashboards can show more than you realize.
