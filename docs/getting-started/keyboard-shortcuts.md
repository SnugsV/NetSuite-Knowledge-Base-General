# NetSuite Keyboard Shortcuts

## Quick Summary

Keyboard shortcuts let you navigate NetSuite faster without reaching for the mouse. The most important shortcut is Ctrl+G (Cmd+G on Mac) for global search, but several others can speed up common tasks.

## Difficulty

Beginner

## Estimated Time

5 minutes

## Overview

NetSuite supports keyboard shortcuts for common navigation and editing actions. These shortcuts work in most modern browsers on Windows and Mac.

Shortcuts are especially useful for:

- Frequent navigation to global search
- Saving records without clicking the Save button
- Moving between fields in forms
- Canceling edits without searching for the Cancel button

## Universal Shortcuts

These shortcuts work throughout NetSuite:

| Action | Windows | Mac |
|---|---|
| **Global search** | Ctrl+G | Cmd+G |
| **Save current record** | Ctrl+S | Cmd+S |
| **Cancel / return** | Esc | Esc |
| **Move to next field** | Tab | Tab |
| **Move to previous field** | Shift+Tab | Shift+Tab |
| **Open selected list item** | Enter | Enter |
| **Refresh page** | F5 | Cmd+R |

## Form Entry Shortcuts

When editing a record or creating a transaction:

| Action | Shortcut |
|---|---|
| Save and continue editing | Ctrl+Shift+S |
| Add new line (in sublists) | Ctrl+Insert |
| Delete selected line (in sublists) | Ctrl+Delete |
| Move to next sublist row | Down Arrow |
| Move to previous sublist row | Up Arrow |

## Navigation Shortcuts

| Action | Shortcut |
|---|---|
| Go to Home dashboard | Alt+H (or click Home) |
| Open menu bar items | Alt+[underlined letter] (browser-dependent) |
| Switch between open tabs | Ctrl+Tab |
| Close current tab | Ctrl+W |

Note: Some browser-level shortcuts (like Alt+letter for menu navigation) may behave differently depending on your browser and operating system.

## Browser Shortcuts That Work in NetSuite

Since NetSuite runs in a browser, standard browser shortcuts also apply:

| Action | Windows | Mac |
|---|---|
| Open new browser tab | Ctrl+T | Cmd+T |
| Close tab | Ctrl+W | Cmd+W |
| Find in page | Ctrl+F | Cmd+F |
| Zoom in | Ctrl++ | Cmd++ |
| Zoom out | Ctrl+- | Cmd+- |

## Global Search Shortcut

The global search shortcut (Ctrl+G / Cmd+G) is the most important shortcut to learn. It works from any page in NetSuite:

1. Press **Ctrl+G**
2. Type a customer name, item name, transaction number, or keyword
3. Use **Arrow Down** to navigate results
4. Press **Enter** to open the selected record

This replaces many menu navigation steps with a single shortcut.

## Common Mistakes

- **Expecting all apps to share the same shortcuts**: NetSuite shortcuts are specific to the application. They may not match other ERP or business software.
- **Not using the global search shortcut**: Clicking the search field manually is much slower than Ctrl+G.
- **Assuming shortcuts work in all browsers**: Some less common browsers may not support all shortcuts. Use a modern browser like Chrome, Edge, or Firefox for best results.

## Best Practices

- Learn **Ctrl+G / Cmd+G** first — it saves the most time.
- Use **Tab** instead of clicking to move between fields when entering data.
- Use **Ctrl+S** to save frequently — it is faster than finding the Save button on long forms.
- If a shortcut does not work, check whether another browser add-on or extension is intercepting it.
