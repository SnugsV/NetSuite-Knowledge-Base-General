## Recommended Reading Order

For a brand-new NetSuite user, follow this order:

1. [What Is NetSuite?](what-is-netsuite.md) — understand the platform
2. [NetSuite Editions](netsuite-editions.md) — know which features are available
3. [User Interface Overview](user-interface.md) — see the interface
4. [Navigation](navigation.md) — learn to get around
5. [Centers and Roles](centers-and-roles.md) — understand why you see what you see
6. [Dashboards](dashboards.md) — make the dashboard work for you
7. [Global Search](global-search.md) — the fastest way to find anything
8. [Lists and Records](lists-and-records.md) — understand the data model
9. [Personalization](personalization.md) — make NetSuite yours
10. [Keyboard Shortcuts](keyboard-shortcuts.md) — work faster

The [Glossary](glossary.md) is a reference you can return to at any time.
