# NetSuite User Interface Overview

## Quick Summary

The NetSuite user interface is organized around centers, dashboards, menus, and portlets. What you see depends on your role and the center you are using. The interface is web-based and accessed through a browser — no software installation is required.

## Difficulty

Beginner

## Estimated Time

10 minutes

## Overview

NetSuite's interface is a web application that adapts to the user. Unlike traditional desktop software where every user sees the same layout, NetSuite changes what it shows based on the user's role, center assignment, and personalization settings.

The interface has four main structural elements:

1. **Center** — The primary navigation framework (e.g., Home, Sales, Inventory)
2. **Dashboard** — The default landing page that shows portlets
3. **Menu Bar** — Top navigation with role-specific menu items
4. **Portlets** — Widget-like panels on the dashboard

## The Center Framework

A center is a collection of related pages, menus, and dashboards organized around a business function. The most common centers include:

| Center | Purpose |
|---|---|
| Home | Default landing, personal dashboard, recent records, reminders |
| Sales | Sales orders, quotes, customer records, sales reporting |
| Inventory | Items, locations, transfers, adjustments |
| Purchasing | Purchase orders, vendors, receiving |
| CRM | Leads, prospects, customers, campaigns |
| Reports | Saved searches, financial reports, SuiteAnalytics |
| Setup | Company preferences, users, roles, customization |

Users are assigned to one or more centers through their role. A sales representative might see the Home, Sales, and CRM centers, while a warehouse worker might see Home and Inventory.

## The Dashboard

Every center has a dashboard that serves as its landing page. The Home Center dashboard is typically the first page you see after logging in.

A dashboard contains **portlets** — resizable panels that display specific information. Common portlet types include:

- **Reminders** — Orders needing approval, opportunities closing soon, overdue tasks
- **Recent Records** — Last 10 records you viewed or edited
- **Key Performance Indicators (KPIs)** — Metrics like sales this month, open orders, cash balance
- **Custom Portlets** — Saved searches, charts, reports embedded on the dashboard

Dashboards can be customized per user. See [Personalization](personalization.md) for details.

## The Menu Bar

The top menu bar is the primary way to navigate between centers, transactions, lists, reports, and setup pages. Menu items are role-dependent — a user with only basic permissions sees far fewer options than an administrator.

Common menu structure:

```text
[Home] [Transactions] [Lists] [Reports] [Setup] [Documents] [Custom]
```

Each top-level menu opens a dropdown with sub-items. For example:

```text
Transactions > Sales > Enter Sales Order
Transactions > Inventory > Transfer Inventory
Reports > Saved Searches > All Saved Searches
```

## Portlets

Portlets are the building blocks of NetSuite dashboards. Each portlet displays a specific type of information. Users can:

- Add or remove portlets from their dashboard
- Resize portlets
- Rearrange portlet layout
- Configure portlet settings (filters, display options)

Common built-in portlets include calendars, KPIs, reminders, recent records, and report snapshots. You can also create custom portlets from saved searches.

## The Footer Bar

The bottom of the NetSuite interface shows:

- Your current role
- Account name
- Version and release information
- Support links

## Common Mistakes

- **Looking for features in the wrong center**: If you cannot find a menu option, check your role permissions and center assignment.
- **Ignoring the center structure**: Working in the Sales center when you need inventory data is inefficient. Switch centers rather than searching.
- **Not personalizing the dashboard**: The default dashboard may not show what matters most to your role.

## Best Practices

- Learn which center contains the transactions you use most often.
- Add portlets for the saved searches and KPIs you check daily.
- Use the Home dashboard as your command center by removing unnecessary portlets.
- Remember that if you cannot find a menu option, it may be due to your role, not the interface.
