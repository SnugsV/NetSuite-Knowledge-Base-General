# NetSuite Navigation

## Quick Summary

Navigating NetSuite is primarily done through the menu bar, global search, and center switches. Understanding how to move between centers, find records, and access transactions is the first practical skill a new user should learn.

## Difficulty

Beginner

## Estimated Time

10 minutes

## Overview

NetSuite provides several ways to navigate the application. The most common methods are:

1. **Menu bar** — Traditional dropdown menus organized by function
2. **Global search** — Quick record search from any page
3. **Center navigation** — Switching between functional centers
4. **Shortcuts** — Browser and application keyboard shortcuts
5. **Recent records** — Quick access to recently viewed items

Each method is useful in different situations. Most experienced users combine all of them.

## The Menu Bar

The menu bar runs across the top of the NetSuite interface. It is the most complete navigation tool because it provides access to every transaction, list, report, and setup page available to your role.

Common menu structure:

- **Transactions** — The primary menu for business transactions (sales orders, purchase orders, inventory transfers, invoices, etc.)
- **Lists** — Master record lists (employees, customers, vendors, items, etc.)
- **Reports** — Built-in reports, saved searches, financial reports, and SuiteAnalytics
- **Setup** — Account configuration, preferences, users, roles, customization (usually limited to administrators)
- **Documents** — File cabinet, document templates, template management
- **Custom** — Custom records, custom transactions, custom menu items added by administrators

Each menu item opens a dropdown. For example, to navigate to inventory transfers:

```text
Transactions > Inventory > Transfer Inventory
```

## Switching Centers

If your role has access to multiple centers, you can switch between them using the center menu or the Home link. Each center provides its own dashboard and menu structure tailored to a business function.

To switch centers:

1. Click **Home** to return to the Home Center dashboard
2. Use the **center selector** if available in your theme
3. Use shortcuts or saved shortcuts configured by your administrator

Switching centers changes the dashboard and sometimes the available menu options. See [Centers and Roles](centers-and-roles.md) for more detail.

## Global Search

The global search field is located in the top-right corner of the NetSuite interface. It is the fastest way to find a specific record — a customer, an item, a transaction, or a saved search.

To use global search:

1. Press **Ctrl+G** (Windows) or **Cmd+G** (Mac), or click the search field
2. Type a keyword — customer name, item name, document number, or record type
3. Select the matching result from the dropdown

See [Global Search](global-search.md) for detailed usage.

## Recent Records

NetSuite tracks the records you have recently viewed. The Recent Records portlet on the Home dashboard shows your last 10 records. You can also access recent records from the menu by clicking the record type name (e.g., hover over Transactions > Sales > Sales Orders to see recent sales orders).

## Navigation by Record Type

A useful pattern for finding related records:

1. Open a record (e.g., a customer record)
2. Use the **Actions** menu or subtabs to navigate to related records (transactions, communications, addresses)

For example, from a customer record you can:

- View all sales orders for that customer
- View all invoices
- View all returns
- View communications and notes

This is often faster than navigating through menus, especially when working on a specific account.

## Common Navigation Patterns

### Finding a Transaction

```text
Transactions > [Module] > [Transaction Type]
```

Example: `Transactions > Sales > Enter Sales Order`

### Finding a List or Record

```text
Lists > [Category] > [Record Type]
```

Example: `Lists > Relationships > Customers`

### Finding a Report

```text
Reports > [Category] > [Report Name]
```

Example: `Reports > Sales > Sales by Customer Summary`

## Common Mistakes

- **Clicking through menus one level at a time**: Use global search or keyboard shortcuts for speed.
- **Assuming a missing menu item means the feature does not exist**: It may simply be restricted by your role.
- **Navigating to the same record through menus every time**: Use recent records, global search, or saved searches.

## Best Practices

- Use **Ctrl+G** (global search) as your primary navigation tool for known records.
- Use the **menu bar** when exploring available transactions or reports.
- **Bookmark** frequently used transactions in your browser or in a saved shortcut.
- **Learn the menu structure** relevant to your role — it helps when you need to find something similar but not identical.
