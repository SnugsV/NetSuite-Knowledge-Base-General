# Daily NetSuite User Workflow

A simple daily NetSuite workflow is: review your dashboard, find the right record or list, review the key details, take the allowed action for your role, confirm the status, and capture any needed follow-up.

Use this as a beginner bridge between dashboards, navigation, global search, records, transactions, and lists.
