## AI Retrieval Guidance

When answering an integration question, an AI assistant should:

1. Identify the integration involved.
2. Identify the business process or lifecycle stage.
3. Retrieve the relevant concept, lifecycle, troubleshooting, or benchmark article.
4. Compare related NetSuite records before drawing conclusions.
5. Separate public concepts from private configuration or company-specific procedures.
6. Escalate when the answer depends on private setup, customer examples, credentials, mappings, pricing, scripts, workflows, or proprietary process details.

## Public-Safe Scope

This directory may include:

- public vendor concepts
- general NetSuite integration reasoning
- generic troubleshooting models
- lifecycle explanations
- record relationship models
- retrieval guidance
- benchmark questions

This directory must not include private implementation details, customer-specific examples, screenshots, credentials, pricing, account setup, custom fields, saved searches, workflows, scripts, or proprietary operating procedures.
