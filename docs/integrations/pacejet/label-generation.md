# Label Generation

## Quick Summary

Label generation creates the physical shipping label for each package. Pacejet supports standard carrier labels, custom labels through the label connector, and batch label printing.

## Difficulty

Intermediate

## Estimated Time

10 minutes

## Overview

The shipping label is the final output of the rate shopping process. Once a carrier and service are selected, Pacejet generates the label in the format required by the carrier and makes it available for printing.

## Label Types

| Label Type | Description | Use Case |
|---|---|---|
| **Standard carrier label** | Generated by Pacejet in carrier format | Simple operations, single carrier |
| **Custom label (Label Connector)** | Designed in BarTender with custom fields | Branded packaging, multi-carrier |
| **Batch labels** | Multiple labels generated in one process | High-volume fulfillment |

## Standard Labels

Pacejet generates standard carrier labels for all supported carriers. Labels include:

- Carrier logo and service level
- Ship-to address
- Package weight and dimensions
- Tracking number and barcode
- Special handling instructions

## Custom Labels (Label Connector)

For companies with branded packaging or specific label requirements, the Pacejet Label Connector integrates with BarTender label software to generate custom labels. The label connector:

- Maps Pacejet CSV fields to BarTender label fields
- Supports customer-specific label templates (trading partners)
- Allows testing before production deployment

## Label Printing

Labels can be printed:

- **Individually** — Each label prints as the shipment is processed
- **In batch** — All labels for a batch of shipments print together
- **On schedule** — Labels generated and printed at scheduled intervals
