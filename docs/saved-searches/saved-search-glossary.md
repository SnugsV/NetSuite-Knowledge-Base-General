# Saved Search Glossary

## Quick Summary

A reference of terms used in Saved Searches. Use this glossary when you encounter an unfamiliar term in other articles.

## A

**Available Filter**
A user-facing filter that allows a search consumer to narrow results without editing the search. Configured in the Available Filters tab.

## C

**Criteria**
The conditions that determine which records appear in search results. Criteria are defined in the Criteria tab.

**CSV Export**
A feature that enables downloading search results as a comma-separated value file for use in spreadsheet applications.

## D

**Dashboard Portlet**
A Saved Search displayed on a NetSuite dashboard as a live, updating data panel.

## F

**Field**
A specific piece of information on a record (e.g., Customer Name, Date Created, Total).

**Filter**
See Available Filter.

**Formula Column**
A result column that calculates its value from other fields using expressions and functions.

## G

**Group By**
A configuration option that organizes results by a common value, showing one row per unique value.

## H

**Highlighting**
Conditional formatting that applies visual emphasis (color, bold, italic) to rows matching specific conditions.

## J

**Join**
A connection from the primary search type to a related record type, making fields from the related record available in results and criteria.

## M

**Main Line**
A filter specific to transaction searches that controls whether results show one row per transaction (Yes) or one row per line item (No).

## O

**OR Condition**
A criteria grouping that requires a record to match any of the specified conditions, rather than all conditions.

## R

**Record Type**
The type of record being searched (Transaction, Item, Customer, Employee, etc.). Determines available fields and joins.

**Relative Date**
A date criterion that adjusts automatically (e.g., "This Month," "Last 30 Days," "Today") rather than using a fixed date.

**Result Columns**
The fields displayed in search output. Configured in the Results tab.

**Row Limit**
The maximum number of rows returned by a search.

## S

**Saved Search**
A reusable, configurable query that searches NetSuite records, displays selected columns, and can be saved and shared.

**Search Type**
The record type on which the search operates. Determines available criteria, results, and joins.

**Summary Search**
A search that groups records and calculates aggregated values (sums, counts, averages) using Summary Types.

**Summary Type**
A mathematical operation applied to a result column — Sum, Count, Average, Minimum, or Maximum.
