## Integration Points

Saved Searches connect to nearly every part of NetSuite:

| Module | Connection |
|---|---|
| **Administration** | Permissions control who can create and share searches |
| **Dashboards** | Saved searches can be displayed as portlets |
| **Workflows** | Saved searches can trigger workflows |
| **SuiteScript** | Searches are created and executed programmatically |
| **SuiteTalk** | Search results can be pulled via REST APIs |
| **Inventory** | Stock monitoring, reorder alerts |
| **Sales** | Pipeline tracking, order monitoring |
| **Reporting** | Saved searches feed into financial reporting |
| **SuiteAnalytics** | Saved searches differ from workbooks |
