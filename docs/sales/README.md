# Sales

This module covers the Order-to-Cash (O2C) lifecycle — how organizations convert customer demand into fulfilled orders, revenue, and cash. Sales connects to Inventory, Purchasing, Accounting, and Manufacturing.

## The Order-to-Cash Flow

```
Lead → Quote → Sales Order → Credit Check → Pick → Pack → Ship → Invoice → Payment → Revenue Recognition
```

## How Sales Connects

| Module | Connection |
|---|---|
| **Inventory** | Sales orders reserve and consume inventory at fulfillment |
| **Purchasing** | Drop-ship and special orders create POs from sales orders |
| **Accounting** | Invoices generate AR, payments affect cash, COGS from fulfillment |
| **Manufacturing** | Sales demand drives production schedules |
| **Saved Searches** | Pipeline monitoring, order tracking, customer analysis |
| **Workflows** | Order approval, credit check automation |
| **SuiteScript** | Custom pricing, order automation, integrations |
