# Contribution Guide

## Purpose

This guide explains how content should be added to the NetSuite Knowledge Base.

## Content Statuses

- Draft: Work in progress and not ready for AI use.
- Review: Ready for a reviewer to validate.
- Approved: Reviewed and ready for AI/search use.
- Deprecated: Replaced by newer content.
- Archive: Historical content kept for reference.

## Review Rules

Before a document is marked Approved, confirm that:

1. The page uses the standard template.
2. The content is original and not copied wholesale from Oracle documentation.
3. Official sources are linked.
4. The topic is filed in the right folder.
5. The metadata is complete.
6. The document clearly separates general NetSuite behavior from company-specific practices.

## AI Readiness Checklist

- Clear title
- AI Summary section completed
- Source links included
- Common mistakes included where useful
- Related topics included
- Status set correctly
