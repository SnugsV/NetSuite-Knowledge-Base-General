# NetSuite Knowledge Base - General

A general-purpose NetSuite knowledge base designed for AI assistants, internal training, onboarding, and documentation projects.

This repository is intended to organize NetSuite concepts into clean, original Markdown notes with references back to official Oracle NetSuite documentation. It should not copy Oracle documentation wholesale. Instead, each page should summarize concepts, explain practical usage, and link to source documentation.

## Goals

- Create a clean, AI-friendly NetSuite reference library.
- Organize NetSuite knowledge by topic and module.
- Keep content source-linked and easy to review.
- Support future company-specific overlays, such as internal SOPs, screenshots, saved searches, and custom fields.

## Suggested Structure

```text
docs/
  basics/
  administration/
  inventory/
  saved-searches/
  suitescript/
  suitetalk/
  release-notes/
governance/
templates/
sources/
```

## Documentation Principles

1. Use original summaries and explanations.
2. Link back to official Oracle documentation.
3. Keep each file focused on one topic.
4. Use consistent metadata and headings.
5. Separate general NetSuite knowledge from company-specific processes.
6. Clearly mark drafts, reviewed content, and approved content.
