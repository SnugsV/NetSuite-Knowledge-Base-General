# AI Agent Guide

## Purpose

This file is the first-stop guide for AI agents working in the NetSuite Intelligence Platform repository.

Use it to understand the repository purpose, public-safe boundaries, startup reading order, and small-change workflow before editing files.

## Repository Identity

This repository is a public-safe NetSuite Intelligence Platform. It is designed to help humans and AI assistants reason through NetSuite concepts, records, workflows, integrations, troubleshooting paths, and escalation boundaries.

It is not a private company SOP repository
