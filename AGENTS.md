# AI Agent Guide

Use this as the first stop for AI agents working in this repository.

## Rules

- Keep the repository public-safe. Do not add company SOPs, private screenshots, custom fields, saved searches, workflows, scripts, pricing, credentials, customer examples, proprietary details, or internal prompts.
- Read `README.md`, `PROJECT_STATUS.md`, `SESSION_REPORT.md`, and `knowledge-engine/README.md` before editing.
- Make one small, reviewable change at a time.
