# Contributing

Thank you for helping improve NetSuite Knowledge Base General.

## Contribution Principles

1. Write original educational content.
2. Do not copy Oracle documentation.
3. Link to official Oracle documentation when helpful.
4. Keep articles beginner-friendly unless clearly marked advanced.
5. Use the article template and style guide.
6. Prefer practical examples and common mistakes over generic descriptions.
7. Keep public documentation generic and free of private implementation details.

## Public-Safe Content

This is a public knowledge base. Contributions should explain general NetSuite concepts, broadly applicable patterns, and public-safe troubleshooting guidance.

Do not include company-specific SOPs, customer examples, custom fields, saved searches, workflows, SuiteScripts, screenshots, pricing, credentials, proprietary processes, or details that reveal how a specific organization operates.

## Suggested Workflow

1. Open an issue describing the article or improvement.
2. Create or edit Markdown using the templates in `/templates`.
3. Confirm the article follows `/governance/STYLE_GUIDE.md`.
4. Add sources to `SOURCES.md` when applicable.
5. Submit a pull request.

## Article Requirements

Every article should include:

- One H1 title
- Quick summary
- Difficulty
- Estimated time
- Prerequisites, if applicable
- Main explanation
- Best practices
- Common mistakes
- FAQ, if useful
- Related articles
- Oracle references

## Copyright and Source Policy

This project should summarize, explain, and teach. It should not mirror, scrape, or republish Oracle documentation. When official documentation is relevant, cite it with a link and write original explanation around it.
