# Ecosystem Domains

## Purpose
Define the external business platforms that interact with NetSuite at a high level. This document establishes scope for ecosystem knowledge without covering implementation details.

## Scope
- ERP ecosystem boundaries
- Platform categorization
- High-level integration context
- Public-safe architectural guidance

## Supported Platforms
- NetSuite
- Pacejet
- SPS Commerce
- Avalara
- EDI platforms
- Shipping carriers
- Middleware and iPaaS platforms
- REST API integrations

## High-Level Interaction with NetSuite
These platforms exchange business information with NetSuite to support processes such as order management, taxation, logistics, fulfillment, electronic document exchange, and system integration. Specific configurations and implementations are documented separately.
