# NetSuite Touchpoints

## Purpose
Define the common, product-neutral interaction points between NetSuite and external platforms. This document provides a consistent reference for how ecosystem applications exchange information with NetSuite.

## Scope
This document identifies interaction points only. It does not describe configuration, implementation, troubleshooting, or platform-specific behavior.

## Common Touchpoints
- Records (customers, vendors, items, transactions)
- Business events and workflows
- REST APIs
- SOAP web services
- CSV import and export
- SuiteTalk integrations
- SuiteScript integrations
- Middleware and iPaaS connections
- EDI document exchange
- Scheduled data synchronization

## Typical Information Exchange
- Customer data
- Item and inventory data
- Sales orders
- Purchase orders
- Shipments and fulfillment
- Invoices
- Tax information
- Status updates
