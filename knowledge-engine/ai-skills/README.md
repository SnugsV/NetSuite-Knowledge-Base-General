# AI Skills

This directory contains executable AI Skills — structured reasoning guides that teach an AI how to perform specific tasks the way an experienced NetSuite consultant would.

## Purpose

Skills are the next evolution of the repository. While the ERP Curriculum teaches what things are, and the Intelligence Frameworks teach how to think about problems, Skills teach how to execute tasks.

## Available Skills

|---|---|
| [Saved Search Builder](saved-search-builder.md) | How to design, build, and validate Saved Searches |

## How to Use a Skill

1. Read the skill document to understand the full reasoning framework
2. Follow the decision trees to gather information
3. Use the prompt patterns and interview frameworks to collect customer input
4. Execute the build process
5. Validate against the checklist
6. Document the reasoning and alternative approaches
