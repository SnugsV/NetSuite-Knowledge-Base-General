# Customer Context Intelligence Framework

This framework teaches an AI how to understand a customer's unique NetSuite environment before attempting to solve a problem. Every customer implementation is different. The AI must learn to discover those differences systematically.

## Purpose

Experienced ERP consultants rarely begin with an answer. They begin by understanding the customer. This framework codifies that process — what information matters, why it matters, and how to gather it.

## The Context Discovery Process

```
Initial Contact
       ↓
Classify the Problem
       ↓
Gather Company Profile
       ↓
Discover NetSuite Configuration
       ↓
Inventory Customizations
       ↓
Request Metadata (if needed)
       ↓
Assess Solution Confidence
       ↓
Recommend → or → Escalate
```
